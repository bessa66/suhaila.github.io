The Internet of Things(IoT) consists of sensors or devices that transmitted data through an internet. While Cloud services stores data on remote servers over the Internet. The data is taken from a sensor using Raspberry Pi4, and stored in Cloud using ThingSpeak and MongoDB.

System Overview

Raspberry Pi4 has been used to transmit data from sensor to Cloud services. Raspberry Pi is The Raspberry Pi is a low cost, credit-card sized computer that plugs into a computer monitor or TV and uses a standard keyboard and mouse.   The Raspberry Pi has the ability to interact with the outside world and has been used in a wide array of digital maker projects. It runs a Linux-based operating system and is equipped with built-in Wi-Fi, Bluetooth, and GPIO (General Purpose Input/Output) pins, making it ideal for connecting and controlling sensors, actuators, and other devices.
